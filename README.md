EasyOCR Image Text Reader
A simple project that uses EasyOCR to extract text from images. EasyOCR is a robust and lightweight Optical Character Recognition (OCR) tool, capable of recognizing text in over 80 languages with high accuracy.

Features
Detects and reads text from images.
Supports multiple languages for diverse applications.
Efficient and easy-to-integrate OCR solution.

Requirements
Python 3.6 or higher
EasyOCR library
PyTorch (as required by EasyOCR)
